cloud_download Connecting with Web Data Connector

If you're seeing this message you need to access the URL using Tableau's Web Data Connector.

Copy the URL from your web browser https://beta.tableaumapping.bi/wdc-legacy, open Tableau Desktop, in the connections list select 'Web Data Connector' and paste the URL into the connector. Then it's just like a regular Tableau data source.

You'll need Tableau 10.1, 10.2 or 10.3. If you've got 10.4 or newer why not use our latest connector at https://beta.tableaumapping.bi/wdc

 layers Building Your Map

If you're using an older version of Tableau, don't worry we've got you covered! You'll just need to build your map using the polygon mark. Here's how:

  1. Change the mark type to Polygon
  2. Drag the dimenion -> Path to the Path drop zone
  3. Drag both -> Polygon ID and -> Sub Polygon ID onto level of detail
  4. Either double click -> Latitude or drag it to the Rows shelf
  5. Either douple click -> Longitude or drag it to the Columns shelf
