cloud_download Connecting with Web Data Connector

If you're seeing this message you need to access the URL using Tableau's Web Data Connector.

Copy the URL from your web browser https://beta.tableaumapping.bi/wdc, open Tableau Desktop, in the connections list select 'Web Data Connector' and paste the URL into the connector. Then it's just like a regular Tableau data source.

You'll need Tableau 10.4 or newer. If you have 10.1, 10.2 or 10.3 you can use our legacy connector https://beta.tableaumapping.bi/wdc-legacy (give it a second, you should be auto forwarded)
