cloud_download Connecting with Web Data Connector

If you're seeing this message you need to access the URL using Tableau's Web Data Connector.

Copy the URL from your web browser http://alpha.tableaumapping.bi/wdc, open Tableau Desktop, in the connections list select 'Web Data Connector' and paste the URL into the connector. Then it's just like a regular Tableau data source.

You'll need Tableau 10.2 or newer.
