Preview Map

As you may notice the TableauMapping.bi blog format has been replaced by an interactive Mapbox map. This is intended to preview the boundary data available and help you choose what you want to download in to Tableau.

To start simply navigate to the location you're interested in by either panning the map or use the navigator in the top left. Then click the + button in the bottom right of the screen to see boundaries available in that location.

Selecting a boundary type will load it on the map. It's worth noting that the preview data is more generalised than the data that's passed to Tableau and so gaps may be visible between boundaries.

Data to Tableau

Of course playing with the preview map is great, but at some point you're probably going to want to bring the data down into Tableau. This is done via the Web Data Connector.

Simply create a new connection in Tableau Desktop, choose the Web Data Connector, then in the URL box enter http://beta.tableaumapping.bi/wdc

You should then be presented with a list of tables to choose from, each one representing a boundary from the preview map. For more info on building the map in Tableau please see our help section
